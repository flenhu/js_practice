Exercise 6d

Imagine an amusement park that a discount for children (6 years and younger) or seniors (65 years and older).

  • create a variable called 'age' and save a person's age inside.
  • create a if-statement to check if the person qualifies for a discount. If they do, display 'Discount' in the console. Otherwise, display 'No discount'.
  • Note: try using the || operator in your solution.
  • Try changing the 'age' variable to display different messages.

Example age: 5
